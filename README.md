## Installation

Install the program:

```bash
go install github.com/kraasch/gochess@latest
```

Get the package:

```bash
go get github.com/kraasch/gochess
```

## Usage

Use the program:

```bash
make build
./build/gochess --help
```

Use the package:

```go
import (
  "github.com/kraasch/gochess"
)

gochess.DoSomething("Hello")
```
